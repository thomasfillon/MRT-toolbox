The MRT toolbox contains the matlab code corresponding to the Time-Frequency Anaysis framework described in :
     
     Thomas Fillon and Jacques Prado, "A Flexible Multi-Resolution
     Time-Frequency Analysis Framework for Audio Signals", in proceedings
     of the 11th International Conference On Information Science, Signal
     Processing and their Applications (ISSPA) 3-5 july 2012, Montreal
     (Canada)

- MRT.m is the main MATLAB class 
- window_properties.m is a utility function to estimate analysis window characteristic

Examples of uses of this toolbox are given in the following files :
- MRT_demo.m  
- MRT_demo_CQT.m	
- MRT_demo_MIDI.m
- MRT_demo_ERB.m
